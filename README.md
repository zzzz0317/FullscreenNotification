# FullscreenNotification
这是我的第一个开源项目，请大家多多支持~

## 功能
* 类似微信强提醒，没有时间限制
* 可选 收到消息后自动点击通知/弹出像微信强提醒类似的界面(后面这个功能暂时有bug)
* 设置后是否全屏显示的主动权在发送者手里(消息中含有关键词)
* 或者全屏显示标题(一般是用户昵称)中含有关键词的通知(秒回npy消息很有用~)

## 使用场景
* 我不管，npy发来消息我就是要第一时间打开回复
* 替代微信强提醒功能，不用反复打开开关，适用于所有应用
